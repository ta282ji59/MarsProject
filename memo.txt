・tarファイルのDockerImageを解凍する方法
    docker load -i ファイル名.tar

・PostGISのDockerコンテナに入る方法（意図：データベースの中身を更新、確認するため）
    - Dockerコンテナに入る
    docker exec -it redace1_postgis bash

    - psqlプロンプトに入る
    psql -U postgres

    - [🚨初期設定]SQLコマンドを実行(ユーザーsakaiがデータベースmarsおよびそのスキーマ内のすべてのテーブル、シーケンス、および関数にアクセスできるようになる)
    GRANT ALL PRIVILEGES ON DATABASE mars TO sakai;
    GRANT ALL PRIVILEGES ON ALL TABLES IN SCHEMA public TO sakai;
    GRANT ALL PRIVILEGES ON ALL SEQUENCES IN SCHEMA public TO sakai;
    GRANT ALL PRIVILEGES ON ALL FUNCTIONS IN SCHEMA public TO sakai;

    - Dockerコンテナを閉じる
    exit
